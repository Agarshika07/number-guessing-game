# Number Guessing Game (Python)

A simple Python number guessing game with limited attempts and helpful clues to guide the player.

## How the Game Works
- The computer randomly selects a number between 1 and 100
- The player has **3 chances** to guess the number
- After each wrong guess, a clue is given (higher or lower)
- If all chances are used, the correct number is revealed

## Features
- Random number generation
- Limited attempts
- Hint system (higher / lower)
- Clean and simple game logic

## How to Run
Make sure Python is installed, then run:

```bash
python guessing_game.py
